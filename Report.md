$$\sum_{i=1}^n i = \frac{n(n+1)}{2}$$

```matlab
nx = 0:5;                   % interval of x
x = [1 1 1 1 1 1];          % x[n]
figure, stem(nx, x)         % plot x[n]
y = conv(x, x);             % y[n]
ny = 0:10;                  % interval of y, set a value for "?"
figure, stem(ny, y)         % plot y[n]
```
