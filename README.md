my-markdown-dialect
===================
[dirkarnez/printing-friendly-html-code-blocks](https://github.com/dirkarnez/printing-friendly-html-code-blocks)

### Elements to support
- [ ] `import` code files
- [x] code blocks with line number
- [x] math blocks
    - 2 `$` for centered `$$\sum_{i=1}^n i = \frac{n(n+1)}{2}$$`
    - 1 `$` for left `$$\sum_{i=1}^n i = \frac{n(n+1)}{2}$$`
- [ ] `import` [JSXGraph](https://jsxgraph.uni-bayreuth.de/home/)
- [x] explicit newlines
  - ```html
    1<br>2
    ``
- [x] explicit newlines
  - ```html
    1<hr>2
    ``
- [ ] image and caption
- [ ] quotes
- [ ] footnotes
